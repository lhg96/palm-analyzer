# Screenshots 가이드

이 폴더에는 Palm Analyzer 웹 서비스의 주요 화면 스크린샷들이 포함되어 있습니다.

## 필요한 스크린샷 목록

다음 이미지들을 촬영하여 이 폴더에 저장해주세요:

### 1. main-interface.png
- **내용**: 메인 페이지의 전체 화면
- **포함 요소**: 
  - 네비게이션 바
  - 제목 및 설명
  - 카메라 섹션과 결과 섹션이 나란히 배치된 화면
  - 서비스 소개 섹션
  - 사용방법 안내

### 2. camera-capture.png
- **내용**: 카메라가 활성화된 상태의 화면
- **포함 요소**:
  - 카메라 미리보기 화면
  - "카메라 중지" 버튼
  - "사진 촬영" 버튼 (활성화 상태)
  - 파일 업로드 섹션

### 3. analysis-result.png
- **내용**: 손금 분석이 완료된 결과 화면
- **포함 요소**:
  - 분석된 손금 이미지 (색상별로 표시된 라인들)
  - 통계 정보 (총 라인 수, 주요 라인, 중간 라인, 세부 라인)
  - 검출된 라인 유형 태그들
  - 처리 시간 정보
  - 다운로드 버튼

### 4. detailed-stats.png
- **내용**: 분석 결과의 상세 통계 부분 확대
- **포함 요소**:
  - 라인 개수별 통계 (숫자와 레이블)
  - 라인 타입별 분류 태그들
  - 처리 시간 정보

## 스크린샷 촬영 가이드

### 권장 사항
- **브라우저**: Chrome 또는 Safari 사용
- **화면 크기**: 데스크톱 해상도 (1920x1080 이상)
- **파일 형식**: PNG (투명도 지원)
- **품질**: 고해상도 (텍스트가 선명하게 보일 것)

### 촬영 방법
1. 웹 서비스를 http://localhost:8000 에서 실행
2. 각 기능을 실제로 사용하면서 스크린샷 촬영
3. 손바닥 이미지는 테스트용 샘플 이미지 사용 권장
4. 브라우저 개발자 도구는 닫은 상태로 촬영

### 파일명 규칙
- 정확히 위의 파일명을 사용해주세요
- 소문자와 하이픈(-) 사용
- .png 확장자 사용

## 테스트용 손바닥 이미지

분석 테스트를 위해서는 다음과 같은 손바닥 이미지를 사용하는 것을 권장합니다:
- 손바닥이 선명하게 보이는 이미지
- 적절한 조명이 있는 환경에서 촬영된 이미지
- 손가락보다는 손바닥 부분이 중심인 이미지
- 배경이 단순한 이미지
